#jquery-linkit jQuery Plugin#

**jquery-linkit** is a simple jQuery plugin to attach links to elements. 

##Version##
0.2.0

##Usage##
			$('span').linkMe({
				href: 'http://test.com',
				text: 'click me', 
				target: '_blank'
			});

##Vendors##
jQuery - [http://jquery.com](http://jquery.com)
